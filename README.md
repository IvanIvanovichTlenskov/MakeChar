# MakeChar - Stable Diffusion Extension

Extension for Stable Diffusion WebUI that adds a tab with text input field.

## Installation

1. Go to the "Extensions" tab in Stable Diffusion WebUI
2. Click "Install from URL"
3. Paste this repository URL
4. Click "Install"
5. Restart WebUI

## Features

- Adds a new tab with text input field
- Place for future extensions and features
